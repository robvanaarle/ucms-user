# UCMS User
Usermanagement module for Ultimo MVC

## Requirements
* PHP 5.3
* Ultimo MVC Config
* Ultimo MVC ORM
* Ultimo MVC Form
* Ultimo MVC ACL
* Ultimo MVC Phptpl
* Ultimo MVC Translate
* Ultimo MVC Validation

## Features
* Login / logout
* List, create, update and delete users by admin
* Update username and password by members
