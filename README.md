# UCMS User
Usermanagement module for Ultimo MVC

## Requirements
* PHP 5.3
* Ultimo Config MVC
* Ultimo ORM MVC
* Ultimo Form MVC
* Ultimo ACL MVC
* Ultimo Phptpl MVC
* Ultimo Translate MVC
* Ultimo Validation MVC

## Features
* Login / logout
* List, create, update and delete users by admin
* Update username and password by members
